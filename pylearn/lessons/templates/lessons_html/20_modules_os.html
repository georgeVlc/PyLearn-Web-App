Welcome to Python Modules: Mastering Useful Built-in Modules

This chapter will explore some of Python's most useful built-in modules. These modules provide ready-to-use functionality that simplifies common programming tasks, helping you write efficient and clean code. The first module we’ll cover is os, a powerful tool for interacting with your operating system.

What is the os Module?

The os module allows you to interact with the operating system in a platform-independent way. You can perform tasks like creating directories, listing files, retrieving environment variables, and more.

Common Functions in the os Module

Here are some key functions provided by the os module:

  • os.getcwd(): Returns the current working directory.
  • os.listdir(path): Lists all files and directories in the given path.
  • os.mkdir(path): Creates a new directory at the specified path.
  • os.remove(path): Deletes the file at the specified path.
  • os.environ: Provides access to environment variables.

Example: Using os to Manage Directories

Let’s look at an example that demonstrates some common os module functions:

# Importing the os module
import os

# Get the current working directory
cwd = os.getcwd()
print("Current Directory:", cwd)

# Create a new directory
os.mkdir("new_folder")
print("new_folder created!")

# List all files and directories
contents = os.listdir(".")
print("Directory Contents:", contents)

# Remove the created directory
os.rmdir("new_folder")
print("new_folder removed!")

Using Environment Variables

You can use os.environ to access or modify environment variables:

# Access an environment variable
import os
home_directory = os.environ.get("HOME")
print("Home Directory:", home_directory)

# Set a new environment variable
os.environ["MY_VAR"] = "Hello"
print("MY_VAR:", os.environ["MY_VAR"])

Conclusion

The os module is an essential tool for interacting with your operating system. Whether you’re managing files, directories, or environment variables, os provides the functionality you need. In the next lesson, we’ll explore the sys module and learn how to interact with system-specific parameters.
