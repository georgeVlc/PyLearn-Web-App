Key Concept: Classes in Python

A class in Python is a blueprint for creating objects. Each object can have attributes (data) and methods (functions).

class Example:
    def __init__(self, name):
        self.name = name
    def display_name(self):
        print(self.name)

In this lesson, you'll create a Task class that defines what a "task" is and how it behaves.

Task Class Structure

Below is the initial structure for your Task class:

class Task:
    def __init__(self, title, description, priority, deadline):
        self.title = title
        self.description = description
        self.priority = priority
        self.deadline = deadline
        self.completed = False

    def mark_as_complete(self):
        self.completed = True

This class includes attributes like title, description, priority, and deadline, as well as a method to mark a task as complete.

Task

Implement the Task class as shown above. Once done, extend it by adding a method to display the task details in a user-friendly format.

def display(self):
    print(f"Title: {self.title}")
    print(f"Description: {self.description}")
    print(f"Priority: {self.priority}")
    print(f"Deadline: {self.deadline}")
    print(f"Completed: {self.completed}")
