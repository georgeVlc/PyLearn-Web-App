Lesson 6: Marking Tasks as Complete

In this lesson, you will learn how to mark tasks as complete in the Task Management System and filter tasks based on their completion status. This feature helps in organizing tasks and tracking progress.

What Will You Learn?

By the end of this lesson, you will be able to:

  • Update the completed attribute of a task to True.
  • Filter and display completed and incomplete tasks separately.

Marking Tasks as Complete

We will implement a function named mark_task_complete() to update the status of a specific task. This function will:

  • Search for a task by its title or ID.
  • If found, set its completed attribute to True.
  • If not found, display an appropriate error message.

Code Template

Here is a basic structure for the mark_task_complete() function:

def mark_task_complete(task_list, task_id):
    for task in task_list:
        if task.title == task_id:  # Replace with task ID if applicable
            task.mark_as_complete()
            print(f\"Task '{task_id}' marked as complete.\")
            return True
    print(f\"Error: Task with title '{task_id}' not found!\")
    return False

In the code above:

  • The function searches for a task with the given title or ID.
  • If the task exists, it calls the mark_as_complete() method on the task object.
  • If the task does not exist, an error message is displayed.

Filtering Tasks by Completion Status

To filter tasks, we will implement a function named filter_tasks_by_status(). This function will:

  • Take a list of tasks and a status (e.g., True for completed or False for incomplete).
  • Return a list of tasks matching the specified status.

Code Template

Here is a basic structure for the filter_tasks_by_status() function:

def filter_tasks_by_status(task_list, status):
    return [task for task in task_list if task.completed == status]

In the code above:

  • The function uses a list comprehension to filter tasks based on the completed attribute.
  • It returns the filtered list.
