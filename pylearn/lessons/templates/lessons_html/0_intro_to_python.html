Introduction to Python

Python is a popular, high-level programming language that is easy to read and write. It is widely used in fields such as web development, data analysis, artificial intelligence, scientific computing, and more.

Why Python?

  • Easy to learn and use
  • Open source and community-driven
  • Highly versatile, with libraries for almost every domain

Python Syntax Basics

Python uses simple, clean syntax that emphasizes readability. Let's go over a few basic concepts:

Printing Output

You can use the print() function to display text on the screen. For example:

print("Hello, World!")

Variables

Variables are used to store data. You don't need to declare their type explicitly:

name = "Alice"  # A string
age = 25          # An integer
pi = 3.14159      # A float

Data Types

Common Python data types include:

  • int: Integers (e.g., 1, 2, 3)
  • float: Floating-point numbers (e.g., 3.14, 2.71)
  • str: Strings (e.g., "Hello")
  • bool: Booleans (e.g., True, False)

Comments

Use the # symbol to add comments in your code. Comments are ignored by Python:

# This is a comment
print("Comments are useful!")

Indentation

Python uses indentation to define blocks of code. For example, a for loop:

for i in range(5):
    print(i)  # Indented block
Note: Proper indentation is critical in Python. If you don't indent correctly, your code will raise an error.

Basic Input

Use the input() function to get input from the user:

name = input("What is your name? ")
print("Hello, " + name + "!")
