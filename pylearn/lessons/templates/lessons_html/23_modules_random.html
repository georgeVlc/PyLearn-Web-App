Exploring the random Module in Python

The random module in Python provides tools for generating random numbers, selecting random items, shuffling data, and more. It’s widely used in simulations, games, and testing scenarios.

Key Features of the random Module

Here are some of the most commonly used functions in the random module:

  • random.random(): Returns a random float between 0.0 and 1.0.
  • random.randint(a, b): Returns a random integer between a and b (inclusive).
  • random.choice(sequence): Returns a random item from a sequence (e.g., list, tuple).
  • random.shuffle(sequence): Shuffles a sequence in place.
  • random.sample(sequence, k): Returns a list of k unique items randomly chosen from a sequence.
  • random.uniform(a, b): Returns a random float between a and b.

Example: Generating Random Floats

The random.random() function generates a random float between 0.0 and 1.0:

# Generate a random float
import random

random_float = random.random()
print(f"Random float: {random_float}")

Example: Generating Random Integers

The random.randint(a, b) function generates a random integer between two specified values:

# Generate a random integer
import random

random_int = random.randint(1, 10)
print(f"Random integer: {random_int}")

Example: Shuffling a List

You can use random.shuffle to shuffle a list in place:

# Shuffle a list
import random

my_list = [1, 2, 3, 4, 5]
random.shuffle(my_list)
print(f"Shuffled list: {my_list}")

Example: Picking Random Items

Use random.choice to select a random item or random.sample to pick multiple unique items:

# Picking random items
import random

my_list = ['apple', 'banana', 'cherry', 'date']
random_item = random.choice(my_list)
print(f"Random item: {random_item}")

random_sample = random.sample(my_list, 2)
print(f"Random sample: {random_sample}")
