Lesson 2: Implementing Task Creation

In this lesson, you will learn how to implement a function to create new tasks in the Task Management System. We will also focus on validating user input to ensure that important fields such as title and priority are provided correctly.

What Will You Learn?

By the end of this lesson, you will be able to:

  • Create a function to add new tasks to the task list.
  • Validate user input to ensure the task has all required fields.
  • Handle missing or invalid inputs and provide helpful error messages.

Task Creation Function

To create tasks, we will write a function called add_task(). This function will:

  • Take input from the user for task details such as title, description, priority, and deadline.
  • Ensure that the title and priority fields are provided (non-empty).
  • Add the task to the task list if all fields are valid.

Task Addition Code Template

Here is a basic structure for the add_task() function:

def add_task(title, description, priority, deadline, task_list):
    if not title or not priority:
        print("Error: Title and priority are required fields!")
        return None
    task = Task(title, description, priority, deadline)
    task_list.append(task)
    return task

In the code above:

  • The function checks if the title and priority are provided. If they are missing, it shows an error message.
  • If the fields are valid, it creates a new task and appends it to the task_list.
