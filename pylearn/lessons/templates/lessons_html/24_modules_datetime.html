Exploring the datetime Module in Python

The datetime module in Python allows you to work with dates and times in a variety of ways. It provides tools to manipulate dates, times, and even perform date arithmetic.

Key Features of the datetime Module

The datetime module has several classes and methods to work with dates and times:

  • datetime.datetime: Represents both the date and time.
  • datetime.date: Represents a date (year, month, day) without time.
  • datetime.time: Represents a time without a date.
  • datetime.timedelta: Represents a difference between two dates or times (i.e., duration).
  • datetime.now(): Returns the current date and time.
  • datetime.today(): Returns the current local date without the time component.
  • datetime.strftime(): Converts a datetime object to a string with a specified format.

Example: Getting the Current Date and Time

The datetime.now() function returns the current date and time:

# Get current date and time
import datetime

current_datetime = datetime.datetime.now()
print(f"Current date and time: {current_datetime}")

Example: Working with Date Only

If you want to work with the date only, you can use the date class:

# Get current date
import datetime

current_date = datetime.date.today()
print(f"Today's date: {current_date}")

Example: Formatting Dates and Times

You can format a datetime object into a string using strftime:

# Formatting date and time
import datetime

current_datetime = datetime.datetime.now()
formatted_datetime = current_datetime.strftime("%Y-%m-%d %H:%M:%S")
print(f"Formatted date and time: {formatted_datetime}")

Example: Date Arithmetic

You can perform date arithmetic using timedelta to add or subtract time:

# Date arithmetic using timedelta
import datetime

current_datetime = datetime.datetime.now()
one_week_later = current_datetime + datetime.timedelta(weeks=1)
print(f"One week later: {one_week_later}")
