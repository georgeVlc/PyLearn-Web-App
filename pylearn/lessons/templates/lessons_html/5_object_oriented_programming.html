Object-Oriented Programming in Python

Learn about the principles of Object-Oriented Programming (OOP) and how they are implemented in Python.

What is OOP?

Object-Oriented Programming is a programming paradigm that uses objects and their interactions to design applications and programs. In OOP, data is represented as objects, and actions are represented as methods.

Key Concepts

1. Classes

Classes are blueprints for creating objects. They define a type and the properties (attributes) and behaviors (methods) of objects:

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def greet(self):
        return "Hello, my name is " + self.name

Example Usage

p = Person("Alice", 25)
print(p.greet())  # Output: Hello, my name is Alice

2. Objects

An object is an instance of a class. Objects are created using class definitions:

p = Person("Alice", 25)

3. Attributes

Attributes are characteristics or properties of an object. They are defined in the classâ€™s `__init__` method:

class Car:
    def __init__(self, make, model):
        self.make = make
        self.model = model

4. Methods

Methods are functions defined inside a class. They operate on the attributes of the object:

class Car:
    def __init__(self, make, model):
        self.make = make
        self.model = model
    
    def display_info(self):
        return f"Car make: {self.make}, Model: {self.model}"

5. Inheritance

Inheritance allows one class to inherit the properties and methods of another class:

class ElectricCar(Car):
    def __init__(self, make, model, battery_size):
        super().__init__(make, model)
        self.battery_size = battery_size
