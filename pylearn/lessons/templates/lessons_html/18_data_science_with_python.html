Data Science with Python

Learn about the role of Python in data science, the tools and libraries used, and common tasks performed in data science projects.

Introduction to Data Science

Data science involves extracting knowledge and insights from structured and unstructured data using various methods, tools, and algorithms.


import pandas as pd
import numpy as np

# Creating a DataFrame
df = pd.DataFrame({
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
})

# Basic operations with pandas
print(df.head())
print(df.describe())

Pandas and Numpy

Pandas is a powerful library for data manipulation and analysis. It provides data structures like DataFrames, which are perfect for structured data.

Data Visualization

Matplotlib and Seaborn are popular Python libraries used for creating visual representations of data.


import matplotlib.pyplot as plt
import seaborn as sns

# Simple plot using Matplotlib
plt.plot(df['Age'], df['Name'], 'o-')
plt.title('Age vs Name')
plt.xlabel('Age')
plt.ylabel('Name')
plt.show()

# Using Seaborn for a better visualization
sns.pairplot(df, hue='City')
plt.show()
