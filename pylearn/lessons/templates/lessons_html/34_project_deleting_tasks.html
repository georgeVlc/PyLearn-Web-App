Lesson 5: Deleting Tasks

In this lesson, you will learn how to safely remove tasks from the task list in the Task Management System. This involves finding the specific task to delete and handling cases where the task is not found.

What Will You Learn?

By the end of this lesson, you will be able to:

  • Search for a task in the task list by its title or ID.
  • Safely remove the task from the list if it exists.
  • Handle cases where the specified task does not exist.

Deleting Tasks

To delete a task, we will implement a function called delete_task() that takes the task's identifier (e.g., title or ID) as input and removes the task from the task list if it exists.

Task Deletion Code Template

Below is a basic structure for the delete_task() function:

def delete_task(task_list, task_id):
    for task in task_list:
        if task.title == task_id:  # Replace with task ID if applicable
            task_list.remove(task)
            print(f\"Task '{task_id}' has been deleted.\")
            return True
    print(f\"Error: Task with title '{task_id}' not found!\")
    return False

In the code above:

  • The function iterates over the task list to find a task with the matching title or ID.
  • If the task is found, it is removed from the list, and a success message is displayed.
  • If the task is not found, an error message is shown.
