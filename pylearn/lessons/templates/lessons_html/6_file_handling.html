File Handling in Python

Learn how to handle files in Python. File handling allows you to read from and write to files.

Opening Files

To open a file in Python, use the built-in `open()` function:

# Opening a file in read mode
file = open('example.txt', 'r')

# Opening a file in write mode
file = open('example.txt', 'w')

# Opening a file in append mode
file = open('example.txt', 'a')

Modes

  • `'r'`: Read mode (default). Opens a file for reading only
  • `'w'`: Write mode. Opens a file for writing only, truncating the file first
  • `'a'`: Append mode. Opens a file for writing, but adds to the end of the file
  • `'b'`: Binary mode. Opens the file in binary format
  • `'x'`: Create mode. Creates a new file, returns an error if the file exists

Reading from a File

To read data from a file, use the `read()` method or read the file line by line:

# Reading the entire file
content = file.read()

# Reading a single line
line = file.readline()

# Reading all lines into a list
lines = file.readlines()

Writing to a File

To write data to a file, use the `write()` method:

# Writing to the file
file.write("This is a new line of text.\n")
