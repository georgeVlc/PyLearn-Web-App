Advanced File Handling in Python

Dive into more advanced file handling techniques in Python, including file manipulation, reading/writing CSVs, and working with binary files.

Working with Files

Python provides several functions and methods for handling files, such as open(), read(), write(), close(), and context managers.


# Opening a file
with open('file.txt', 'r') as file:
    content = file.read()
    print(content)

# Writing to a file
with open('file.txt', 'w') as file:
    file.write("Hello, world!")

Reading and Writing CSV Files

CSV files can be read using the csv module in Python, which provides the reader and writer classes.


import csv

# Writing to a CSV file
with open('data.csv', 'w', newline='') as file:
    writer = csv.writer(file)
    writer.writerow(['Name', 'Age', 'City'])
    writer.writerow(['Alice', 30, 'New York'])
    writer.writerow(['Bob', 25, 'Los Angeles'])

# Reading from a CSV file
with open('data.csv', 'r') as file:
    reader = csv.reader(file)
    for row in reader:
        print(row)

Binary File Handling

Handling binary files involves reading and writing data in binary format, such as images, audio, and video files.


# Writing a binary file
with open('image.png', 'wb') as file:
    file.write(b'\x89PNG\r\n\x1a\n...')
    
# Reading a binary file
with open('image.png', 'rb') as file:
    data = file.read()
    print(data[:10])  # Show the first 10 bytes
