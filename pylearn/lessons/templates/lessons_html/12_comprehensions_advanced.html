Comprehensions in Python (Advanced)

Learn advanced usage of list, set, and dictionary comprehensions in Python to simplify and optimize your code.

List Comprehensions

List comprehensions provide a concise way to create lists. They are commonly used for transforming and filtering elements.


# Example of a list comprehension
squared_numbers = [x**2 for x in range(10)]
print(squared_numbers)  # Output: [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Advanced Features

  • Conditional list comprehensions: [x for x in range(10) if x % 2 == 0] to filter even numbers.
  • Nested comprehensions: [x**2 for x in [1, 2, 3] for x in [1, 4, 9]] for multiplying all combinations.

Set Comprehensions

Set comprehensions are similar to list comprehensions but produce sets, which are unordered collections of unique elements.


# Example of a set comprehension
unique_squares = {x**2 for x in range(5)}
print(unique_squares)  # Output: {0, 1, 4, 9, 16}

Dictionary Comprehensions

Dictionary comprehensions allow creating dictionaries in a very compact way. They are ideal for transforming lists into key-value pairs.


# Example of a dictionary comprehension
squared_dict = {x: x**2 for x in range(5)}
print(squared_dict)  # Output: {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Advanced Features

  • Transforming multiple lists: {k: v for k, v in zip(keys, values)}.
  • Conditional dictionary comprehensions: {k: v for k, v in data.items() if v > 10}.
