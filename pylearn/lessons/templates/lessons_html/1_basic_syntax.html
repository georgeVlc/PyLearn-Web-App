Basic Syntax in Python

Learn the fundamental syntax elements of Python, such as variables, comments, and basic operators.

Variables and Assignment

In Python, variables are used to store data. Unlike many other languages, Python does not require you to specify the type of a variable upon declaration.

x = 5        # An integer
name = "Alice"   # A string

Comments

Comments are used to explain code and make it easier to read. They are ignored by Python:

# This is a comment

Basic Operators

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • %: Modulus
  • **: Exponentiation

Example Usage

result = (10 + 5) * 2    # Basic arithmetic operations
