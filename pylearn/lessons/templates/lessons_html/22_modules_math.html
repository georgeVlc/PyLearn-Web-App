Exploring the math Module in Python

Python’s math module provides a rich collection of mathematical functions and constants that make it easy to perform complex mathematical computations.

Key Features of the math Module

Here are some of the most useful features offered by the math module:

  • math.sqrt(x): Returns the square root of x.
  • math.pow(x, y): Raises x to the power of y.
  • math.pi and math.e: Provides the constants π (pi) and e (Euler's number).
  • math.factorial(x): Returns the factorial of x.
  • math.log(x, base): Calculates the logarithm of x to the specified base (default is base e).
  • math.sin(x), math.cos(x), math.tan(x): Trigonometric functions.

Example: Calculating the Square Root

Here’s a simple example using math.sqrt:

# Importing the math module
import math

# Calculate the square root
number = 16
result = math.sqrt(number)
print(f"The square root of {number} is {result}")

Example: Using Constants

You can use mathematical constants like π and e for calculations:

# Working with math constants
import math

# Calculate the circumference of a circle
radius = 5
circumference = 2 * math.pi * radius
print(f"Circumference of the circle: {circumference}")

Example: Factorial and Logarithms

Here’s an example of calculating a factorial and a logarithm:

# Factorial and logarithm
import math

# Factorial
num = 5
factorial = math.factorial(num)
print(f"Factorial of {num} is {factorial}")

# Logarithm
log_value = math.log(100, 10)
print(f"Logarithm base 10 of 100 is {log_value}")

Conclusion

The math module provides a comprehensive set of tools for mathematical computations. Whether you need basic operations like square roots and powers or advanced functions like trigonometry and logarithms, math has you covered.
