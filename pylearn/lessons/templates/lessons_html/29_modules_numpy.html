Working with NumPy in Python

In this final lesson of the "Modules" chapter, we will explore NumPy, one of the most important third-party modules in Python. NumPy provides support for arrays, matrices, and a large collection of mathematical functions that are useful for numerical computations.

What is NumPy?

NumPy is a fundamental package for numerical computing in Python. It introduces the ndarray (N-dimensional array) object, which allows for fast and efficient storage and manipulation of large datasets. NumPy also provides a collection of mathematical functions that can be used for operations on arrays.

Installing NumPy

To install NumPy, use pip, the Python package installer:

pip install numpy

Creating NumPy Arrays

To create a NumPy array, we use the numpy.array() function:

# Importing NumPy
import numpy as np

# Creating a 1D array (vector)
arr = np.array([1, 2, 3, 4, 5])
print(arr)  # Output: [1 2 3 4 5]

# Creating a 2D array (matrix)
arr_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(arr_2d)
# Output:
# [[1 2 3]
#  [4 5 6]]

Array Operations

Once we have NumPy arrays, we can perform a variety of mathematical operations on them:

# Element-wise addition
arr = np.array([1, 2, 3])
result = arr + 5
print(result)  # Output: [6 7 8]

# Element-wise multiplication
result = arr * 3
print(result)  # Output: [3 6 9]

# Array multiplication (matrix multiplication)
arr_2d = np.array([[1, 2], [3, 4]])
result = arr_2d.dot(arr_2d)
print(result)
# Output:
# [[ 7 10]
#  [15 22]]

Array Reshaping

NumPy provides methods to reshape arrays, which can be useful for machine learning and data manipulation tasks:

# Reshaping a 1D array to a 2D array
arr = np.array([1, 2, 3, 4, 5, 6])
reshaped_arr = arr.reshape(2, 3)
print(reshaped_arr)
# Output:
# [[1 2 3]
#  [4 5 6]]

Commonly Used NumPy Functions

Here are some commonly used NumPy functions:

  • np.arange(start, stop, step) - Creates an array with evenly spaced values.
  • np.linspace(start, stop, num) - Generates an array of evenly spaced numbers over a specified range.
  • np.mean(arr) - Computes the mean (average) of the elements in the array.
  • np.std(arr) - Computes the standard deviation of the array.
  • np.sum(arr) - Returns the sum of the array elements.

Example: Creating an Array with arange

# Create an array with values from 0 to 10 (exclusive), with step size of 2
arr = np.arange(0, 10, 2)
print(arr)  # Output: [0 2 4 6 8]

Example: Calculating the Mean and Standard Deviation

# Calculating the mean and standard deviation of an array
arr = np.array([1, 2, 3, 4, 5])
mean = np.mean(arr)
std_dev = np.std(arr)

print("Mean:", mean)  # Output: 3.0
print("Standard Deviation:", std_dev)  # Output: 1.4142135623730951
