Functions in Python

Learn how to define and use functions in Python. Functions are reusable blocks of code that perform a specific task.

Defining Functions

Functions are defined using the `def` keyword followed by the function name and parentheses. You can include parameters inside these parentheses.

def greet(name):
    return "Hello, " + name

Example Usage

def add(a, b):
    return a + b
print(add(2, 3))  # Output: 5

Parameters and Arguments

Functions can accept multiple arguments, which can be positional or keyword arguments.

def multiply(x, y):
    return x * y

print(multiply(4, 5))  # Positional arguments
print(multiply(x=4, y=5))  # Keyword arguments

Returning Values

Functions can return values using the `return` keyword:

def square(n):
    return n ** 2

Example Usage

result = square(3)  # result is 9

Function Scope

Variables defined inside a function are local to that function and do not affect variables outside:

def modify_list(lst):
    lst.append(4)
    return lst

numbers = [1, 2, 3]
modify_list(numbers)
print(numbers)  # Output: [1, 2, 3, 4]
