Data Structures in Python

Learn about the basic data structures available in Python such as lists, tuples, sets, and dictionaries.

Lists

Lists are ordered, mutable, and allow duplicate elements:

my_list = [1, 2, 3, 'a', 'b']

Common Operations

  • append(): Adds an element to the end of the list
  • extend(): Adds all elements of a list to the end of another list
  • remove(): Removes a specified element from the list
  • pop(): Removes an element at a specified position
  • index(): Returns the index of the first occurrence of a value
  • count(): Returns the number of times a value appears in the list

Tuples

Tuples are ordered and immutable, meaning they cannot be changed after creation:

my_tuple = (1, 2, 'a', 'b')

Common Operations

  • count(): Returns the number of times a value appears
  • index(): Returns the index of the first occurrence of a value

Sets

Sets are unordered collections of unique elements:

my_set = {1, 2, 3, 'a', 'b'}

Common Operations

  • add(): Adds an element to the set
  • remove(): Removes a specified element
  • discard(): Removes an element if it exists
  • pop(): Removes a random element
  • union(): Returns the union of two sets
  • intersection(): Returns the intersection of two sets

Dictionaries

Dictionaries are collections of key-value pairs:

my_dict = {'name': 'Alice', 'age': 25}

Common Operations

  • keys(): Returns a list of all keys in the dictionary
  • values(): Returns a list of all values in the dictionary
  • items(): Returns a list of key-value tuples
  • get(): Returns the value associated with a key
  • update(): Updates the dictionary with another dictionary
  • pop(): Removes a key-value pair
