Lesson 4: Editing Tasks

In this lesson, you will learn how to edit specific tasks in the Task Management System. You will modify attributes such as title, description, priority, deadline, and status.

What Will You Learn?

By the end of this lesson, you will be able to:

  • Access and modify attributes of a specific task in the task list.
  • Write a function that updates a task based on user input.
  • Handle errors gracefully if the task is not found.

Editing Tasks

To edit a task, we will need to implement a function that takes the task's identifier (e.g., title or ID) and allows the user to modify its attributes.

Task Editing Code Template

Below is a basic structure for the edit_task() function:

def edit_task(task_list, task_id, title=None, description=None, priority=None, deadline=None, completed=None):
    task = next((t for t in task_list if t.title == task_id), None)  # Find task by title (or ID if applicable)
    if not task:
        print(f"Error: Task with title '{task_id}' not found!")
        return None

    # Update attributes if provided
    if title:
        task.title = title
    if description:
        task.description = description
    if priority:
        task.priority = priority
    if deadline:
        task.deadline = deadline
    if completed is not None:
        task.completed = completed

    return task

In the code above:

  • The function checks if the task exists using a generator expression that looks for a task with a matching title (or ID).
  • If the task is found, it updates the attributes that were passed as arguments.
  • If the task is not found, it displays an error message.
