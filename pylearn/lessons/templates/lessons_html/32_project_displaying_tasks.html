Lesson 3: Displaying Tasks

In this lesson, you will learn how to format and display tasks in a user-friendly way. You will also learn how to sort tasks based on different attributes like priority and deadline.

What Will You Learn?

By the end of this lesson, you will be able to:

  • Format and display tasks in a readable format.
  • Sort tasks based on priority or deadline.
  • Implement a filtering mechanism to view tasks based on status or other attributes.

Displaying Tasks

To display tasks in a user-friendly way, we will use the display() method in the Task class that we created earlier. However, we will extend this functionality to list multiple tasks.

Task Display Code Template

Below is a template for displaying a list of tasks:

def display_tasks(task_list):
    for task in task_list:
        task.display()

The display_tasks() function will iterate through a list of tasks and call the display() method on each task, printing its details.

Sorting Tasks

We will also learn how to sort tasks based on attributes like priority or deadline. Here is a basic template for sorting:

def sort_tasks(task_list, key='priority'):
    return sorted(task_list, key=lambda task: getattr(task, key))

This function takes a list of tasks and a sorting key (e.g., 'priority' or 'deadline') and returns the list sorted by the specified attribute.
